      Contribute material

      Tip: for PDFs, upload to Google Drive, set sharing to "Anyone with the link", then paste that link here.
      To actually prevent download/print (not just hide the buttons), open the file in Drive → Share → the settings/gear icon → turn off "Viewers and commenters can download, print, and copy". Without this, Drive's own preview will still show those buttons — that's a Drive setting, not something we can override from here.
      Each line = one page, top to bottom. Right-click-save on an image can never be fully prevented in a browser — this just removes any dedicated download button from view.
